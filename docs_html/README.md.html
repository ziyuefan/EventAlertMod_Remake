開關型指令，用於顯示設定齒輪顯示與否，加上重設強制定位定位到小地圖左下角 /eam UpdateInterval n

設定更新頻率，越小越快，若有團戰無法疲勞狀況請加大此數值，預設0.1S（0.1 ~ 1秒）

/eam IconAppenSpellTip

開關形指令、圖示是否在滑鼠移開時顯示指令指令

/eam ShowRunesBar

開關型指令，用於決定是否顯示死騎符文列(2020/10/18新增)

Command line

The following instructions are case-insensitive:

/eam SCDRemoveWhenCooldown

Switch completion command, the skill cooling icon will be removed after cooling down (true means to remove; false means not to remove)

/eam SCDNocombatStillKeep

Switch type command, whether the skill cooling icon will still be displayed after leaving combat (true will remain displayed; false will not be displayed after leaving combat)

/eam SCDGlowWhenUsable Switch type command, whether the skill cooldown is highlighted when available (true means highlighted when available, false otherwise) This instruction specifically states that it uses IsUsableSpell() to determine if the instruction is due to resources or If there is a debuff that prevents it from being cast but its skill conditions cannot be met, it will not be highlighted and the distance is not limited

/eam MiniMap [Reset]

Switch type command, used to display whether the setting gear is displayed or not, plus reset the forced positioning to the lower left corner of the minimap /eam UpdateInterval n

Set the update frequency. The smaller it is, the faster it is. If you are unable to get tired in a team battle, please increase this value. The default is 0.1S (0.1 ~ 1 second)

/eam IconAppenSpellTip

Whether switch-shaped commands and icons are displayed when the mouse is moved away

/eam ShowRunesBar

Switch type command, used to decide whether to display the death knight rune column (new in 2020/10/18)

/eam StackFontSize nSize

Command to change the upper data size (case-insensitive, you can also use /eam sfs nSize instead)

/eam TimerFontSize nSize * To change the number size (not case sensitive, you can also use /eam tfs nSize instead)*

/eam SNameFontSize nSize

  • To change the role name size (case-insensitive, you can also use /eam nfs nSize instead)*

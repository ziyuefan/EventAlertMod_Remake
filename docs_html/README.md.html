以下指令說明不分大小寫:

/eam SCDRemoveWhenCooldown

開關型指令,技能冷卻圖示在冷卻完成後移除(true表示要移除;false表示不移除)

/eam SCDNocombatStillKeep

開關型指令,技能冷卻圖示在脫離戰鬥後是否仍顯示(true保持顯示;false 脫戰不顯示)

/eam SCDGlowWhenUsable

開關型指令,技能冷卻是否在可用時高亮(true表示可用時高亮false則否) 這指令特別說明其使用IsUsableSpell()判斷,也就是如果該指令因為資源或 有減益導致無法施放或者無法滿足其技能條件,則不會高亮但是距離不在此限

/eam MiniMap [reset]

開關型指令,用來顯示設定齒輪顯示與否,加上 reset 強制定位定位到小地圖左下角

/eam UpdateInterval n

設定更新頻率,越小越快,若有團戰無法負荷情況請加大此數值,預設 0.1S (0.1 ~ 1秒)

/eam IconAppenSpellTip

開關形指令, 圖示是否在滑鼠移過時顯示法術說明

/eam ShowRunesBar

開關型指令,用來決定是否顯示死騎符文列(2020/10/18新增)

/eam StackFontSize nSize

指令以改變堆疊層數大小(不分大小寫,亦可以/eam sfs nSize代替)

/eam TimerFontSize nSize

以改變計時數字大小(不分大小寫,亦可以/eam tfs nSize代替)

/eam SNameFontSize nSize

以改變法術名稱大小(不分大小寫,亦可以/eam nfs nSize代替)

Command Line

The following instructions are case-insensitive:

/eam SCDRemoveWhenCooldown

Switch type command, the skill cooling icon will be removed after the cooling is completed (true means to remove; false means not to remove)

/eam SCDNocombatStillKeep

Switch type command, whether the skill cooling icon will still be displayed after leaving combat (true will remain displayed; false will not be displayed after leaving combat)

/eam SCDGlowWhenUsable Switch type command, whether the skill cooldown is highlighted when available (true means highlighted when available, false otherwise) This instruction specifically states that it uses IsUsableSpell() to determine, that is, if the instruction is due to resources or If there are debuffs that prevent it from being cast or its skill conditions cannot be met, it will not be highlighted but the distance is not limited

/eam MiniMap [reset]

Switch type command, used to display whether the setting gear is displayed or not, plus reset to force positioning to the lower left corner of the minimap /eam UpdateInterval n

Set the update frequency. The smaller it is, the faster it is. If there is a team battle that cannot handle the load, please increase this value. The default is 0.1S (0.1 ~ 1 second)

/eam IconAppenSpellTip

Switch-shaped command, whether the icon displays the spell description when the mouse is moved over

/eam ShowRunesBar

Switch type command, used to determine whether to display the death knight rune column (newly added on 2020/10/18)

/eam StackFontSize nSize

Command to change the stacking layer size (not case sensitive, you can also use /eam sfs nSize instead)

/eam TimerFontSize nSize To change the timing number size (not case sensitive, you can also use /eam tfs nSize instead)

/eam SNameFontSize nSize

To change the size of the spell name (not case sensitive, you can also use /eam nfs nSize instead)

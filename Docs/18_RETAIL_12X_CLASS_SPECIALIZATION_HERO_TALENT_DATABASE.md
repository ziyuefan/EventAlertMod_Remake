#EventAlertMod 12.x 午夜職業專精與英雄天賦極限實戰資料庫(12.0.7)

本文件詳細記載了EventAlertMod（EAM）在魔獸世界服正式12.0.7版本中，所支援的**13個職業、39個專精**，以及**28個英雄天賦**的核心監控配置與實戰校對指標。此資料作為AddOn本地化選項篩選、一鍵載入預設值以及未來AI代理維護裝備庫時的權威參考資料。

## 🛠️ 監控配置欄位屬性解釋

在庫配置中，除了常用的`id`（武器ID）與`type`（武器監控類型，如`aura`、`spellCooldown`）之外，我們也為實戰優化配置了下列進階屬性：

* **`unit`**：監控的單位目標，通常為`"player"`（玩家本身）或`"target"`（當前目標），寵物為`"pet"`。
* **`pandemicTime`**: Pandemic Glow（未知亮框）** 效果提示重新整理。
* **`maxStacksAlert`**：溢位防呆層數。當有層數達到此值時，會以高亮或特殊視窗提醒玩家佔用資源，以防止浪費。
* **`minStacksAlert`**：最低警報層數。當發生層數達到此值時才開始警報（或註意提醒）。
* **`showValue`**：數值監控。主要用於防護吸收量（如戰士無視苦痛、武僧天神酒），會在圖示上動態數值。
* **`glowOnApply`**：施放時發光。當光環應用時立即啟用高亮發光。
* **`grayOnMove`**：移動置灰。當感知到玩家移動且無移動施法Buff時，圖示會降低排水度（Alpha = 0.3），提示無法施放。
* **`moveBuffs`**: 移動施法 Buff 清單。只要玩家身上有此列表中的 Buff（如冰浮、懸空、靈行者之賜），即使在移動中，圖標也不置灰。
* **`burstFilterActive`**：爆發警報標記。當特定的爆發Buff（如燃燒）啟動時，自動點亮此傳染病的流行病或提示低優先級警報，減少視線幹擾。

## ⚡ 資源防溢位與高能防解決策略（資源安全邊際）
在戰鬥中，必須預防核心資源的溢出。 EAM的定義配置了「臨界層數」與「防固定閾值」，在即將資源溢出時給予玩家強烈的發光高亮提示，優先消耗。

1. **法師：秘法 - 虛空精準 (Void Precision, 382445)**
   * **機制**：最大2層。當層數=2時，下一個秘法衝擊/彈幕必消。
- **防呆**：`maxStacksAlert = 2`。當層數達到2時，圖示強烈發光，引導消耗，防止後續大法師之觸因飛彈施放造成層數浪費。
2. **法師：裂空者 - 魔法碎片 (Splinter, 444621)**
   * **機制**：最高至8層自動釋放。
   - **防呆**：`maxStacksAlert = 8`。
3. **薩滿：增強 - 渦渦武器 (Maelstrom Weapon, 344179)**
* **機制**：天賦可升級至10層，但滿10層若不釋放，後續近戰白字與風暴打擊會造成線索渦流失敗。
   - **防呆**：`maxStacksAlert = 10`。到達10樓時，耳機渦流武器圖示轉為特大發光狀態，強制玩家穿插瞬發閃電箭/元素衝擊。
4. **戰士：武器/防護 - 巨像之力 (Colossus Might, 434947)**
   * **機制**：最大10層，提供大量技能傷害增益。
- **防呆**：`maxStacksAlert = 10`, `minStacksAlert = 3`。小於3層時不顯眼，但疊滿10層時啟動爆發高亮，提示玩家進入狂暴斬殺或主動洩力輸出。
5. **術士：惡魔學問 - 惡魔核心 (惡魔核心, 264173)**
   * **機制**：最大可儲存4層。若達到4層而不消耗，小鬼死亡或暴君消失時產生的核心將發生故障損失。
- **防呆**：`maxStacksAlert = 4`。在滿4層時強制高亮，提示立即施放瞬發惡魔之箭。
6. **死亡騎士：血液 - 骸骨之盾 (Bone Shield, 195181)**
   * **機制**：防戰坦核心。需維持在5層以上才能享受完整減傷與骨盾增益。
- **修改防呆**：`minStacksAlert = 5` (12.0.7實戰，原為3)。當玩家骨盾層數低於5層時，圖示觸發高亮頭部（防呆警告），促使玩家施放主機分裂指引。

## 🔇 爆發超載噪音與噪音優先級 (Burst period De-noising)
在爆發期，玩家的GCD極為緊湊。此時，EAM應進入「爆發策略模式」，引發無關的常規冷卻、防禦防護等「雜訊圖示」，而應將視線全部資源集中在核心觸發輸出與短CD爆發技上。

###喚醒與高亮篩選規則（Burst Filtering）

* **法師（火焰-燃燒190319期間）**：
* **隱藏**：隱藏活動炸彈（活彈，117828）的流行病致命引發。因為燃燒期間全為瞬發炎爆術與火焰衝擊，手動補炸彈是嚴重的GCD浪費與DPS流失。
    * **高亮**：強烈高亮「炎爆術！(48107)」與「霜火賦能(444625)」。
* **術士 (痛苦 - 召喚黑眼 205180 / 腐靈魂爛 198590 期間)**：
* **高亮**：將痛楚（Agony）、枯萎術（Wither）的疫情區域顯示放大1.2倍，加上臨界秒數發光調整為紅色臀部。確保爆發期100%覆蓋DoT，乾榨每一點增傷。
* **死亡騎士 (冰霜 - 冰霜之柱 51271 期間)**：
    * **解除**：激發反魔法護目鏡、冰封之韌等生存技能的圖示（移至紫色或隱藏冷卻）。
* **高亮**：大高亮冷酷嚴冬(Remorseless Winter, 196770)與殺戮機器(51270)的瞬間發湮滅。
* **聖騎士 (懲戒 - 復仇之怒 31884 期間)**：
    * **高亮**：將聖天之鎚(431398)與神聖意志(223817)設定為高亮，置中顯示，以引導無損技能插入。

## 🏃 移動走位可以獲得技能動態引導（動態投射和灰顯）
在12.0.7的高動態大祕境與團本戰鬥中，走位期間的施法判定是區分頂尖遠程/治療玩家的分水嶺。 EAM的設計實作了「行動動態引導」：
當有玩家**移動狀態**（`GetUnitSpeed("player") > 0`），且身上**沒有移動施法Buff**（如法師「浮冰」、薩滿「靈行者之賜」、喚能師「懸空」）時：
* **讀條技能**：自動置灰（Alpha = 0.3），提示目前無法施放，避免玩家原地讀條中斷。
* **瞬發技能/觸發技能**：維持100%亮度或高亮，引導玩家在走位時維持GCD運轉。
* **一旦玩家獲得移動施法Buff，或停止移動**：所有技能立即恢復正常。

## ⚖️ 12.0.7 專家會審修改與最佳化解決記錄
在2026-06-06的全體專家聯席會議審定會議中，各職業與玩家代表針對坦克與近戰/遠程生存Buff的監控提出了以下修改，並已100%回寫至`Data/SpellArray.lua`中：

1. **骸骨之盾 (DK 鮮血)**：
   * **決議**：將最少警報層數 `minStacksAlert` 由 3 修改為 **5**。高層實戰中低於 5 層骨盾屬於高危險狀態，必須提早預警。
2. **惡魔尖刺（DH復仇）**：
* **決議**：新增`glowRequiresCharge = true`（手上有充能才提示續杯）與`vacuumAlert`（無尖刺且無充能時發出/黃色防禦空白警告色）。
3. **正義盾擊（聖騎防護）**：
   * **決議**：新增`holyPowerOverflowAlert = 5`（聖能滿5格時強制高亮防溢位），且`glowRequiresHolyPower = 3`時允許續杯發光。
4. **盾牌格擋（戰士防護）**：
* **決議**：新增`vacuumAlert = true`，加上無格擋Buff且格擋充能可用時，彈出紅色斷檔警告。
5. **無視苦痛（戰士防護）**：
   * **決議**：啟用`showPercentOfMaxHP = true`將吸收盾量以最大生命值百分比顯示。當大於85%上限時變暗（防滲過刷），低於10%且怒氣足時微弱觸發。
6. **鐵××（德魯伊守護者）**：
* **決議**：**徹底刪除無效的`pandemicTime = 1.8` **。鐵機制××為高層獨立計時，無法續杯重新整理。改為`showTotalStacks = true`（顯示目前總共有多少層數），並在怒氣 >= 90 且層數 < 2 時啟用`rageOverflowAlert`高亮消怒提示。
7. **荒野荊棘（德魯伊野性）**：
* **決議**修改：`pandemicTime` 為 **1.8 秒**（原為 4.5 秒）。荒野之藤基礎持續 6 秒，30% 臨界值為 1.8 秒，設定 4.5 會導致剛套用即無效亮框。
8. **醉拳(武僧羅賓)**：
* **決議**：引入`dynamicStaggerScaling = true`，醉拳三色渲染不再參考暴雪最大血量射擊，改以「每秒扣血射擊」判定（綠色:<3% | 黃色: 3%~6% | 紅色：>=6%）。天神酒與8層以上淬鏈靈藥實作組合發光金色高亮。
9. **狂風暴雨 (薩滿增強)**：
   * **決議**：新增【風暴使者】英雄天賦核心觸發`狂風暴雨(ID:452201)`，觸發時強制亮起發光。
10. **核心治療HoT雙軌疫情解鎖**：
    * **決議**：後續AuraService重構應解除`isDebuff`限制，使Helpful Aura（如回春術、復甦之霧）也能經歷無損繼承的Pandemic Glow。
